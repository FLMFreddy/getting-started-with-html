How web works

Browser makes a request via internet to the server, the request is made by http protocol. Server then responds with files which the browser displays

HTTP -> the main protocol of the web
Servers -> computers that host files that make up the web
Internet -> the world's largest computer network
Browser -> a program that runs on your computer to display files found on the web

HTML - is made up of:
- text content what you see
- markup what it looks like
- references to other documents (videos, images)
- links to other pages

HTML markup
<name> tag content </name>
Examples:
b tag - <b>content</b>
em tag - <em>content</em>
There are void tags that do not have content like <br> tag which creates a newline in the html

Computers are stupid - you have to tell them precisely what you want them to do.

HTML Attribute - this is a property of an HTML tag like href in anchor tag
